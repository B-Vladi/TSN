<static> TSN.compileFromDir(pattern, path)
Асинхронная рекурсивная компиляция файлов, включая вложенные дирректории. 
  // Компилировать только файлы с расширением .html.
  TSN.compile('html');

  // Аналогично предыдущему вызову.
  TSN.compile(/.+?\.html$/);

  // Компилировать все файлы в папке /home/user и подпапках.
  TSN.compile(null, '/home/user');
Parameters:
{string|RegExp} pattern Optional, Default: /.* /
Расширение файла (строка), либо ругелярное выражение, которому должно соответствовать полное имя файла для компиляции. После завершения компиляции вызывается событие TSN#event:compileDirEnd.
{string} path Optional, Default: TSN.config.templateRoot
Путь к дирректории, в которой необходимо компилировать файлы.
<static> {function} TSN.compileFromFile(path, config)
Синхронная компиляция шаблона из файла. 
  // Компиляция файла /home/user/template.xml.
  TSN.config.templateRoot = '/home/user';
  TSN.compileFromFile('template.xml');

  Компиляция с указанием кастомного конфига.
  var template = TSN.compileFromFile('template.xml', {
    name: 'CustomName',
    removeXMLDeclaration: false
  });
  TSN.cache.CustomName === template // true
Parameters:
{string} path
Путь к файлу шаблона относительно TSN.config.templateRoot.
{object} config Optional
Объект конфигурации шаблона: TSN.config.
Returns:
{function} Скомпилированный шаблон template.

<static> {text} TSN.render(template, context, stream)
Синхронный рендеринг шаблона. 
  TSN.config.templateRoot = '/home/user';

  // Рендеринг шаблона из /home/user/template.xml.
  TSN.render('template.xml', context);

  // Аналогично предыдущему вызову.
  TSN.compileFromFile('template.xml');
  TSN.render('template.xml', context);

  // Аналогично предыдущему вызову.
  TSN.render(TSN.compileFromFile('template.xml'), context);

  // Компиляция с произвольным именем и рендеринг с записью результата в поток.
  TSN.compileFromFile('template.xml', {
    name: 'CustomName'
  });
  TSN.render('CustomName', context, stream);
Parameters:
{string|function} template
Скомпилированный шаблон или строка, представляющая собой имя шаблона или путь, относительно TSN.config.templateRoot. Если по относительному пути шаблон не был скомпилирован - он будет скомпилирован.
{object} context Optional, Default: undefined
Контекст шаблона.
{object} stream Optional, Default: undefined
Поток с возможностью записи, в который будет записываться результат рендеринга.
Returns:
{text} Результат рендеринга.
